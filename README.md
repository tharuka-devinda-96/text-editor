# Text Editor

This is a simple text editor desktop application built using Java SE and JavaFX. It allows users to create, edit, and save text files with basic file I/O concepts. The application also includes a rich text editor using HTML formatting in JavaFX.

## Features

- Create new text files
- Open existing text files
- Edit and save text files
- Rich text editor with HTML formatting
- User-friendly interface

## Requirements

- Java SE Development Kit (JDK)
- JavaFX SDK
- Apache Maven (optional)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/tharuka-devinda-96/text-editor.git
2. Open the project in your preferred Java development environment (e.g., IntelliJ, Eclipse).
3. Enjoy the text editing experience!
- Ensure you have Java SE Development Kit (JDK) installed. You can download it from Oracle's website.

- Set up JavaFX SDK. You can download it from the JavaFX website.

- Build the project using Maven (optional):
   ```bash
  mvn clean package
## Usage

1. Compile and run the application using your preferred IDE or command line.

2. The text editor window will open.

3. Use the menu options or toolbar buttons to perform actions like creating a new file, opening an existing file, editing the text, and saving the file.

4. The text editor supports basic formatting options like bold, italic, underline, etc., using HTML tags.

5. To save the file, use the "Save" option in the file menu options. The file will be saved with a .txt extension.
